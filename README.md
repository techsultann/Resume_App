# Resume_App

This application is written 100% in kotlin and consist of 2 modules for now, a lot of works are in progress to make it perfect.

This design is gotten from dribble link below from FATHAN ASHANUL FARDI
https://dribbble.com/shots/14894936-CV-Mobile-App/attachments/6608074?mode=media

https://github.com/hdodenhof/CircleImageView
